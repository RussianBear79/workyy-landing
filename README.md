# Workyy Landing Page

Landing page для веб-платформы Workyy — аналитических досок с SQL/Python-узлами и совместной работой.

## Технологии

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Установка

```bash
pnpm install
```

## Запуск

```bash
pnpm run dev
```

## Сборка

```bash
pnpm run build
```

## Структура

- `src/components/HomePage.tsx` - главный компонент landing page со всеми секциями
